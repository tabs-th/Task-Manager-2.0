# Task-Manager-2.0

Race gear checklist for runners! 

To Setup:
1. Clone repository to local machine
2. Navigate to project folder: 'cd task-manager-2.0'
3. Install dependencies: 'npm install'
4. Run server: 'npm run dev'
5. Open [http://localhost:3000]

Design Decisions
* Theme: I chose a high-contrast neon-on-black theme to capture the running energy.
* colors: I used Neon Green as the accent color, which mimics high-viz gear and clearly delineates buttons.
* Font: Headers are bolded and italicized serif
* Layout: Centered and used a container to separate the checklist.
